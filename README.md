# Mohammed Al-Yazji React Portfolio

## تشغيل المشروع

```bash
npm install
npm run dev
```

## بناء نسخة الإنتاج

```bash
npm run build
```

## الموجود حاليًا

- React + Vite
- React Router
- عربي / إنجليزي
- Dark / Light mode
- Responsive mobile drawer
- صفحات الخدمات والمشاريع والمدونة والتواصل
- روابط المشاريع الحية
- بدون Firebase حاليًا

## الخطوة القادمة

إضافة Firebase Authentication وFirestore وStorage ولوحة Admin.
